# 리팩토링 실습 (차량조립 KATA)
기존에 작성된 코드를 리팩토링하는 실습이다.
이미 제작된 소스코드가 존재한다.

# 기존에 사용중인 시스템의 아쉬운 점
절차지향식 코드로, 유지보수가 어려운 구조
안전하지 않은 문법들이 사용
확장성이 고려되지 않음
유닛테스트가 없음


# 차량 제조 순서
1) 자동차 타입을 선택한다.
세단, SUV(에스-유-브이), 트럭
2) 자동차에 들어갈 부품을 선택한다.
엔진, 제동장치, 조향장치
3) 완성된 차량을 테스트한다.
선택한 부품이 자동차 타입에 사용 가능한지 검사

## 자동차 타입 선택
차량의 종류를 선택한다.
총 세가지 타입(세단, SUV(에스-유-브이), 트럭)을 제작할 수 있으며,
향후에 타입이 더 추가될 수 있다.

## 자동차 조립
엔진, 제동장치, 조향장치를 각각 선택한다.

## 자동차 완성 가능조합 확인
### 제한조건 1
제동장치에 Bosch 제품을 사용했다면, 조향장치도 Bosch 제품을 사용해야한다.
(타사 제품과 호환되지 않는다.)

### 제한조건 2
 Continental은 Sedan용 제동장치를 만들지 않는다.
(-> 세단에 Continental 제품 사용 불가)
도요타는 SUV용 엔진을 만들지 않는다.
WIA는 Truck용 엔진을 만들지 않는다.
Mando는 Truck용 제동장치(brake System)을 만들지 않는다.
